Normalization (Value/ULN or Value/LLN)

  • Increase analytes (e.g., ALT, AST, BILI): Normalized as Value / ULN
    • Y-axis shows Ratio: green zone 0–1 = normal, >1 = abnormal (elevated)
  • Decrease analytes (e.g., PLAT, NEUT, HGB): Normalized as Value / LLN
    • Y-axis shows % of LLN: ≥100% = normal, <100% = abnormal (decreased)
    • Example: Platelet = 75 with LLN = 150 → 50% of LLN
  • Bidirectional analytes (e.g., K, SODIUM, CA): Use ULN or LLN based on which side is abnormal (unified ratio >1 = abnormal)
  • Value Source: Standard (SI) values first, then Local Lab values as fallback
  • Point Styles: ● Circle = Standard (SI Units), ■ Square = Local Lab Units, ▲ Triangle = Hardcoded (Manual)
  • X-Axis: Study Day (Day 0 = C0D1 or first screening visit)
  • Exclusion: Patients whose ALL visits are within normal range are excluded by default
