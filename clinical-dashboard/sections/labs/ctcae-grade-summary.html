    Worsening from baseline: Post-treatment CTCAE grade > Baseline CTCAE grade

    Each subject counted once per analyte using worst post-treatment value

      - Increased parameters (BILI, ALT, AST, ALP): Maximum post-treatment value

      - Decreased parameters (NEUT, PLAT, GLUC): Minimum post-treatment value

      - Bidirectional parameters (K, SODIUM, CA): Worst post-treatment CTCAE grade

    Baseline: C0D1 visit (if missing, use SCRN)

    ────────────────────────────────────────────────────────────────

    Data Source Logic:

      - CTCAE Grading: Uses Standard (SI) values whenever available

      - Local Lab fallback: Allowed only for ratio-safe analytes (e.g. ALT/AST/BILI/ALP/GGT/CREAT/LIPASE/AMYLASE/INR/APTT/PT/PTT)

      - Absolute-threshold analytes (e.g. GLUC/HGB/CHOL/TRIG/URATE/K/SODIUM/CA): Do not use Local Lab fallback without standardized units

      - Increase analytes (ALT, AST, BILI, ALP, etc.): Require Value + ULN

      - Decrease analytes (NEUT, PLAT, HGB, etc.): Require Value + LLN

      - Bidirectional analytes (K, SODIUM, CA, etc.): Require Value + ULN + LLN

      - Records missing required reference ranges are excluded from analysis
