One-sided Normalization (ULN/LLN)

  • Normal Range (Green Zone): Y-axis values from 0 to 1 represent the normal range for all analytes
  • Increase analytes (e.g., ALT, AST, BILI): Normalized as Value / ULN
    • When Value ≤ ULN → Ratio ≤ 1 → Normal
    • When Value > ULN → Ratio > 1 → Abnormal (elevated)
  • Decrease analytes (e.g., PLAT, NEUT, HGB): Normalized as LLN / Value
    • When Value ≥ LLN → Ratio ≤ 1 → Normal
    • When Value < LLN → Ratio > 1 → Abnormal (decreased)
  • Bidirectional analytes (e.g., K, SODIUM, CA): Use ULN or LLN based on which side is abnormal
  • Key Insight: Both normalization methods result in Ratio ≤ 1 = Normal and Ratio > 1 = Abnormal, enabling unified comparison across all analytes
  • Value Source: Standard (SI) values first, then Local Lab values as fallback
  • X-Axis: Study Day (Day 0 = C0D1 or first screening visit)
  • Hardcoded: Included and shown as triangles (override same day)
  • Deduplication: One value per patient per day (highest ratio kept)
  • Exclusion: Patients whose ALL visits are normal (ratio <= 1) are excluded by default
