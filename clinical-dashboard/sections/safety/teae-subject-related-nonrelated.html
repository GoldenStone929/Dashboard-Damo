Subject-Level Analysis Methodology
Each subject (patient) is counted once per adverse event type (Preferred Term). When a subject experiences the same AE multiple times with different attribute values, the following conflict resolution rules apply:
  • Seriousness: Worst case is shown (Y > N)
  • Toxicity Grade: Highest grade is shown (5 > 4 > 3 > 2 > 1)
  • Relationship: Most related is shown (Related > Possibly Related > Unlikely Related > Not Related)
  • Outcome: Value from the most recent event (by start date) is shown
  • Action Taken: Value from the most recent event (by start date) is shown
